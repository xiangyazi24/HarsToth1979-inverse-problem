# Hárs & Tóth (1979) — On the Inverse Problem of Reaction Kinetics

Modern LaTeX transcription of:

> V. Hárs and J. Tóth, "On the Inverse Problem of Reaction Kinetics,"
> *Colloquia Mathematica Societatis János Bolyai*, vol. 30,
> Qualitative Theory of Differential Equations, Szeged (Hungary), 1979, pp. 363–379.

The original paper was typeset on a typewriter. This repository provides a faithful LaTeX rendition with proper mathematical typesetting and TikZ reaction network diagrams.

## Build

```bash
pdflatex main.tex
pdflatex main.tex   # second pass for references
```

## Contents

The paper gives a necessary and sufficient condition (the "lack of negative cross-effects") for a polynomial ODE system to be realizable as a mass-action kinetic differential equation, and constructively provides the *canonic mechanism*.
